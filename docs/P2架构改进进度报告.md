### ⚪ 改进项 #14: 错误处理精细化 - 0% 完成

**目标**:
将通用的 String 错误替换为具体类型：

```rust
// 当前：
pub enum DrorisError {
    Storage(String),
    Query(String),
    Catalog(String),
}

// 目标：
pub enum DrorisError {
    Storage {
        kind: StorageErrorKind,
        tablet_id: Option<u64>,
        message: String,
    },
    Query {
        kind: QueryErrorKind,
        query_id: Option<QueryId>,
        message: String,
    },
}

pub enum StorageErrorKind {
    TabletNotFound,
    TabletAlreadyExists,
    FlushFailed,
    CompactionFailed,
    SegmentNotFound,
}
```

**影响范围**: `crates/common/src/error.rs` + 所有使用 DrorisError 的代码

**预期完成时间**: 3-4 天

## 关键代码修改示例

### DashMap API 变换

```rust
// RwLock<HashMap> 用法：
let nodes = self.nodes.read().await;
let node = nodes.get(&id)?;
self.nodes.write().await.insert(id, node);

// DashMap 用法：
let node = self.nodes.get(&id).map(|r| r.value().clone())?;
self.nodes.insert(id, node);
```
